# runtrackJs

Exercices d'initiation au JavaScript - Jour 01

## Structure
runtrackJs/
└── jour01/
├── job01/  → Popup "Hello Javascript!" en JavaScript inline
├── job02/  → Popup déplacé dans un fichier script.js externe
├── job03/  → Remplacement du popup par un console.log
├── job04/  → Fonction bisextile(année)
├── job05/  → Fonction afficherJoursSemaines() avec boucle for
├── job06/  → Fonction fizzbuzz() de 1 à 151
├── job07/  → Fonction jourTravaille(date) avec jours fériés 2024
├── job08/  → Fonction sommeNombresPremiers(a, b)
├── job09/  → Fonction tri(numbers, order) - algorithme bubble sort
└── job10/  → Fonction compterVoyelles(phrase)

## Compétences travaillées

- Intégration de JavaScript dans une page HTML
- Utilisation de la console web
- Déclaration et appel de fonctions
- Boucles `for`
- Algorithmie (tri, nombres premiers, détection de dates)

## Lancer les exercices

Ouvrir le fichier `index.html` du job voulu dans un navigateur, puis ouvrir la console (F12 → Console) pour voir les résultats.
