# runtrackJs

Exercices JavaScript - Jour 02 : manipulation du DOM et événements

## Structure
runtrackJs/

jour02/
- job01/  → Récupérer et afficher le contenu d'un élément via citation()
- job02/  → Afficher/masquer un article via showhide()
- job03/  → Compteur de clics via addOne() avec addEventListener
- job04/  → Keylogger a-z avec double saisie si focus dans le textarea
- job05/  → Barre de progression au scroll en footer fixed
- job06/  → Code Konami → page stylisée aux couleurs de La Plateforme_
- job07/  → Toggle dark/light theme via changeTheme()

## Compétences travaillées

- Sélection et manipulation d'éléments du DOM
- Gestion des événements (click, keydown, scroll)
- Création et suppression d'éléments dynamiquement
- Modification des styles CSS via JavaScript

## Lancer les exercices

Ouvrir le fichier `index.html` du job voulu dans un navigateur, puis ouvrir la console (F12 → Console) pour voir les résultats.

## Commits suggérés

- `feat(job01)` : récupération et affichage de citation dans la console
- `feat(job02)` : apparition et disparition d'un article au clic
- `feat(job03)` : compteur de clics avec addEventListener
- `feat(job04)` : keylogger avec gestion du focus
- `feat(job05)` : barre de scroll colorée en footer fixed
- `feat(job06)` : code Konami avec style La Plateforme_
- `feat(job07)` : toggle dark/light theme
